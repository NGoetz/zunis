zunis.models.flows.masking module¶

Module for the generation of masks for coupling cells

n_ary_mask(d, n, offset)¶

Create a n-ary mask with n entries (a list of bool with each nth entry True)

Parameters

  • d (int) – numbers of entries in the mask

  • n (int) – period of the mask

  • offset (int) – offset of the True entries

Returns

True/False mask with each nth entry True

Return type

list of bool

n_ary_mask_strategy(d, n, repetitions)¶

Generate an list of masks using the n-ary periodic strategy: every nth entry is True and each next mask is offset by one position with respect to the previous one. We loop for a given number of repetitions through all possible masks.

Parameters

  • d (int) – dimensionality of the space

  • n – periodicity of the mask

  • repetitions – number of iterations through all possible masks

Returns

list of masks

Return type

list of list of bool
